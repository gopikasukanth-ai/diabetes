# Diabetes Prediction using Machine Learning

## 📌 Project Overview
This project predicts whether a person is diabetic or non-diabetic using Machine Learning.

## 🛠️ Technologies Used
- Python
- Google Colab
- NumPy
- Pandas
- Scikit-learn

## 📂 Dataset
- diabetes.csv

## 🤖 Machine Learning Algorithm
- Support Vector Machine (SVM)

## 📊 Project Workflow
1. Import libraries
2. Load the dataset
3. Data preprocessing
4. Split training and testing data
5. Train the SVM model
6. Predict diabetes
7. Evaluate model accuracy

## 📁 Project Files
- internship - diabetes colab.ipynb
- diabetes.csv
- README.md

## 🎯 Result
The model predicts whether a person is diabetic based on medical information.
